About EstimaStruct

EstimaStruct es una plataforma open source de estimación para ingeniería y arquitectura, diseñada para análisis estructural, detalle técnico y control de costos con trazabilidad.

Funciona como puente entre Revit 2027 y el análisis de precios dentro del ecosistema Estimbot, convirtiendo schedules y datos de obra en partidas auditables y listas para presupuesto real.

Construida con FastAPI, SQLite y Vanilla JavaScript para priorizar portabilidad, transparencia y despliegue simple en equipos técnicos.

  • Estimación por partidas, capítulos CSI e insumos
  • Importación de cantidades desde Revit schedules
  • Base pensada para compartirse, auditarse y descargarse desde GitHub
  • Parte del módulo de automatización de Estimbot
