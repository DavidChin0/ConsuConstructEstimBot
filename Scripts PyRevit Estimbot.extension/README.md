# Scripts PyRevit Estimbot Extension

Extensión formal de pyRevit para Estimbot.

## Incluye

- Revit Keynotes
- Export Schedules
- Update Quantities

## Estructura

```text
Scripts PyRevit Estimbot.extension/
  extension.json
  Estimbot.tab/
    bundle.yaml
    Keynotes.panel/
      bundle.yaml
      Revit Keynotes.pushbutton/
        bundle.yaml
        script.py
    Quantities.panel/
      bundle.yaml
      Export Schedules.pushbutton/
        bundle.yaml
        script.py
      Update Quantities.pushbutton/
        bundle.yaml
        script.py
```

## Uso

Coloca esta carpeta dentro de la ruta de extensiones de pyRevit o apúntala como extensión local.

La carpeta `Scripts PyRevit Estimbot/` queda como paquete fuente/portable del mismo contenido.
